# 🚀 DevSecOps Flask Pipeline Project

A hands-on DevSecOps project demonstrating the complete CI/CD workflow for a Flask web application using modern DevOps tools and security practices.

This project is designed to showcase practical experience with Git, GitHub, Docker, Jenkins, and automated deployment while integrating security into every stage of the software development lifecycle.

---

## 📌 Project Overview

The goal of this project is to build and automate the deployment of a Flask web application using industry-standard DevSecOps practices.

The application will be containerized with Docker, automatically built and tested using Jenkins CI/CD pipelines, and enhanced with security scanning tools to ensure secure software delivery.

---

## 🎯 Project Objectives

- Develop a Flask web application
- Containerize the application using Docker
- Implement Git version control
- Manage source code with GitHub
- Build automated CI/CD pipelines using Jenkins
- Integrate security scanning into the pipeline
- Demonstrate Continuous Integration and Continuous Deployment (CI/CD)
- Follow DevSecOps best practices
